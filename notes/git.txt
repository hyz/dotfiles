### git clone single branch
        git clone -b <branch name> [remote repository address]
    fetch所有remote branch并checkout特定branch。
    你用git branch 看不到是因为它只显示本地分支。 看全部
        git branch -a
    如果只想fetch特定分支
        git clone xxx -b remote-branch --single-branch
